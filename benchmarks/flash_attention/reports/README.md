# Profiling Reports / 性能分析报告

This directory is used to store profiling outputs generated by `nsys` and `ncu`. 
This README serves as a placeholder to ensure the directory is tracked by Git, as profiling tools do not create output folders automatically.

本目录用于存放由 `nsys` 和 `ncu` 生成的性能分析报告。
此 README 仅作为占位符，以确保 Git 能够追踪并创建该文件夹，避免性能分析工具因找不到目标目录而报错。
